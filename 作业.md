### 5.2 理解 state 流转

`aqi_feedback` 表有一个 `state` 字段，值为 0 / 1 / 2。

| state | 含义 |
|-------|------|
| 0 | 已提交，待审核 |
| 1 | 已指派，待检测 |
| 2 | 已检测，流程完成 |

**思考**：如果系统需要新增一个功能——"管理员审核后可以**驳回**反馈，驳回后状态改为 -1"，那么：
1. 数据库需要做哪些改动？
2. 后端需要新增哪个接口？
3. 前端需要在哪个页面新增按钮？
